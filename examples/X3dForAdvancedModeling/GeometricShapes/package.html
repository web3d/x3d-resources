This chapter illustrates design patterns and best practices for authoring different geometries through a collection of interesting 3D shapes.

The open-source Meshlab and X3D-Edit authoring tools helped create many of these examples. Tools of interest for generating mathematical shapes include:
  • Kenneth J.M. MacLean: Geometry
  • Math is Fun: Geometry
  • Matlab
  • Meshlab
  • Vector Visualizer in 3D
  • Virtual Polyhedra
  • Wikipedia: Platonic solid and Geodesic grid
  • Wolfram Mathworld: Geometry
  • X3D-Edit
