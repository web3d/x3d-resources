X3D for Advanced Modeling is a work in progress for demonstrating advanced
Extensible 3D (X3D) Graphics modeling, simulation and visualization techniques.

Archive support includes X3D-Edit authoring tool, X3D Tooltips, X3D Validator,
complete course slidesets and videos, plus this open-source model archive.
Source changes are maintained in version control and bug tracking is online.
