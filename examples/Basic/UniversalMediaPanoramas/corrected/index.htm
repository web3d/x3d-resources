Panoramas

Scroll down to view these panorama textures and demo VRML worlds. Please keep in mind that documentation has not yet been created for "panoramas". Content authors must therefore create their own fallback URLs for panoramas should they choose to use them before documentation is available (due later this month, at which point the entire VRML code entry will be available for each panorama). When documentation is complete each panoramic image will also be assigned its own unique copyright notice and credits. In the meantime, read this notice before using these textures.

Early Access Release Note: The resolution (bit depth) of these panoramas will likely be increased in the final release, yielding higher quality images (currently a lower quality compressed JPEG format is used, resulting in some image artifacts -- a higher quality compression will be used in the final release, meaning your panoramas will look better when the final library ships next month). Please notice that no textures will be renamed or removed: this category is "frozen". In other words, use it! As a Web3D content author you are encouraged to build engaging worlds using these and other categories of textures.
